# CEO Briefing Generator - Agent Skill for Weekly Business Audit

## Overview

Generates comprehensive weekly business and accounting audits with CEO Briefing reports.

## Purpose

Automated weekly audit every Monday morning including:
- Revenue analysis
- Expense tracking
- Profit & Loss
- Cash flow
- Outstanding invoices
- Social media performance
- Bottlenecks identification
- Proactive recommendations

## Usage

```markdown
Generate CEO Briefing for this week
Period: [start_date] to [end_date]

Include:
- Revenue summary
- Expense analysis
- Outstanding invoices
- Social media metrics
- Bottlenecks
- Recommendations
```

## Data Sources

- **Accounting**: Odoo ERP (invoices, payments, expenses)
- **Revenue**: Bank transactions, payment processors
- **Social Media**: Facebook, Instagram, Twitter, LinkedIn APIs
- **Tasks**: Obsidian vault (Needs_Action, Done folders)

## Generation Process

1. Fetch accounting data from Odoo
2. Calculate revenue metrics
3. Analyze expenses
4. Generate P&L statement
5. Review outstanding invoices
6. Aggregate social media metrics
7. Identify bottlenecks
8. Generate recommendations
9. Save to `/CEO_Briefings/`

## Scheduling

```bash
# Cron job for Monday 8:00 AM
0 8 * * 1 uv run python -m src.orchestrator.ceo_briefing
```

## Integration

Works with:
- Odoo MCP (accounting data)
- Social Media MCPs (metrics)
- Dashboard updater
